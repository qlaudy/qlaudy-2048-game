# Qlaudy 2048 Game

## Project Description
Qlaudy 2048 Game is a classic sliding puzzle game where the player combines tiles with the same numbers to reach the 2048 tile. The project is built to practice DOM manipulation, game logic, and frontend development skills.

---

## Live Preview
https://qlaudy.github.io/qlaudy-2048-game/

---

## Technologies Used
- HTML5
- CSS3 / SCSS
- JavaScript (ES6+)
- DOM manipulation
- Game logic implementation
- GitHub Pages (deployment)
