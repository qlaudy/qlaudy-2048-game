# Qlaudy 2048 Game

## Introduction
Welcome to the Qlaudy 2048 Game project — a JavaScript implementation of the classic 2048 puzzle game. The goal of the game is to combine tiles with the same numbers to reach the 2048 tile while achieving the highest possible score.

The project was created to practice JavaScript logic, DOM manipulation, responsive layouts, and frontend development skills.

## Live Preview

## Key Features
- Smooth tile movement and merge animations
- Responsive design for desktop and mobile devices
- Real-time score tracking
- Restart game functionality
- Win and lose game states
- Keyboard controls using arrow keys
- Dynamic game board updates

---

## Challenges
During development, several challenges were solved:

### Tile Movement Logic
Implementing correct tile movement and merge behavior while preventing invalid double merges in a single move.

### Game State Management
Managing board updates, score calculation, win/lose conditions, and random tile generation.

### Responsive Layout
Adapting the game board and UI for different screen sizes while maintaining usability.

### Animation Performance
Creating smooth tile transitions and movement effects without performance issues.

---

## Getting Started

### Clone the repository:
```bash
git clone https://github.com/qlaudy/qlaudy-2048-game.git
cd qlaudy-2048-game
```

### Install dependencies:
```bash
npm install
```

### Run the project locally:
```bash
npm start
```

---

## Usage
After, application will be available in the browser. Use arrow keys to move tiles and combine numbers to reach 2048.
